# PROJECT HISTORY

This directory (`history/`) contains the high-level timeline of the project.
It acts as the structured memory for agents across long iterations.

**Structure:**
- `INDEX.md`: This file.
- `[phase-name].md`: A document detailing what was built in this phase, what decisions were made, and context for the next agent.
- There is always one "current" active phase. Update it as work progresses.
